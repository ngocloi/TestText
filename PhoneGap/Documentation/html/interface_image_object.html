- base64Image

To generate a base64 string (of an ImageObject)

Method to convert an image to the form of base64string from a KEDImage.

Parameters
successCBDefault Success call back function name
errorCBDefault Error call back function name
paramsan input object for which the base64 string is required
Returns
The return value is captured in the 'successCB' function and will have the base64 strign of the image
See also
Check the 'errorCB' method for any failures in case of unexpected behaviour of the method. Generally the error call back would return a string giving the description of the error.
function mySucessCB(imageid){
alert('converted imageid'+imageid);
}
var myImageObject;
ImageObject.prototype.base64Image(mySucessCB,myerrorCB, myImageObject);
