- getDaysRemaining

To get the number of days the License of the SDK is valid.

Method to get the remaining days that the SDK license would be valid to use.

Parameters
successCBDefault Success call back function name
errorCBDefault Error call back function name
Returns
The return value is captured in the 'successCB' function and will be having the number of valid days for a valid license.
See also
Check the 'errorCB' method for any failures in case of unexpected behaviour of the method. Generally the error call back would return a string giving the description of the error.
function mySucessCB(numofdays){
alert(numofdays);
}
function myerrorCB(error){
alert(JSON.stringify(error));
}
License.prototype.getDaysRemaining(mySucessCB,myerrorCB);
- setMobileSDKLicense

To set the License of the native SDK.

Method to set the License of the native SDK.

Parameters
successCBDefault Success call back function name
errorCBDefault Error call back function name
paramsA String variable conting the 'License' to be set to the SDK
Returns
The return value is captured in the 'successCB' for a successful operation, and might return in 'errorCB' for an incomplete/invalid operation. Returns the following values in the corresponding functions successCB : KMC_SUCCESS The license string was set successfully . errorCB : KMC_EV_LICENSING Kofax Mobile SDK detected an invalid license KMC_EV_LICENSE_ALREADY_SET Kofax Mobile SDK license has already been set KMC_EV_LICENSE_EXPIRED Kofax Mobile SDK license has expired
See also
