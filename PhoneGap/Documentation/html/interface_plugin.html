- getLayoutProperties

A getter method for the Layout property.

A method which would return a 'Layout' property used to create objects(of SDK) with certain frame. Set the values of the Layout object such as origin, width and height(& visibility) and pass as argument to create the frame. 'Layout' is a common object and can be used by any class/object.
